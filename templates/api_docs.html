API Docs

API documentation for the XMage API.

All calls are made using the GET-method. Responses are formatted using JSON.

Variables

http://api.xleague.info/variables

Shows various variables used by the application.

Players

http://api.xleague.info/players

Player database

http://api.xleague.info/player/<Name>

Info about individual player

Games

http://api.xleague.info/games

Games database

http://api.xleague.info/game/<ID>

Info about individual game

Vouch requests

http://api.xleague.info/requests

Pending vouch requests
